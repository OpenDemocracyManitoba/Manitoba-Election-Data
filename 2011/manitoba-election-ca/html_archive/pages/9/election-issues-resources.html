The purpose of this page is to provide a comprehensive list of websites from various interests, organizations, and stakeholder groups that promote provincial election issues relating to the upcoming 2011 election.

See the bottom of the page for the search criteria we use in order to find the following websites. More webpages will be posted as they become available.

Website Search Methodology

If you have created or are aware of an election issue webpage that is not featured here, please feel free to contact us and we will provide a link to this website. Manitobaelection will not post websites that promote violence, hatred, or intolerance towards any identifiable group.

The following search criteria were used in finding the webpages posted above:

  • A search through the first ten pages of Google search results using the following search criteria: Manitoba Election 2011

  • A search through the first ten pages of Google search results using the following search criteria: Manitoba Election issues

  • A search through the first three pages of Google search results through the following search criteria: Manitoba Election 2011 (name of each topic area contained within each department name of the Manitoba government). For example, for the department ‘Manitoba Food, Agriculture and Rural Initiatives' the following searches would be made: Manitoba election 2011 food, Manitoba election 2011 agriculture, Manitoba election 2011 rural.
